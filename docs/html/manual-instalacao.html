Manual de Instalação

Guia completo para instalar e configurar a Rede de Serviços Comunitários do zero

📦 Pré-requisitos

Instale os programas abaixo antes de começar:

🟢

Node.js 18+

Runtime JavaScript

nodejs.org →
🐙

Git

Controle de versão

git-scm.com →
🔥

Firebase CLI

Linha de comando Firebase

via npm (ver abaixo)
📱

Expo CLI

Só se for rodar o mobile

via npm (ver abaixo)

Após instalar o Node.js, instale as CLIs:

npm install -g firebase-tools
npm install -g expo-cli   # só para mobile

Verifique se está tudo ok:

node --version      # deve mostrar v18.x ou superior
npm --version
git --version
firebase --version

📥 1. Clonar o Repositório

git clone https://github.com/victoralbertoms-wq/rede-servicos-comunitarios.git
cd rede-servicos-comunitarios

1

Instalar dependências

cd web

Rodar em desenvolvimento

✅ Abra o navegador em http://localhost:3000

Build de produção (opcional)

Os arquivos ficam em web/dist/.

🔥 4. Firebase CLI

Autenticar no Firebase

firebase login

Vai abrir o navegador para login com a conta Google do projeto.

Implantar regras do Firestore

⚠️ Este comando precisa ser executado sempre que as regras do Firestore forem alteradas. 
cd firebase
firebase deploy --only firestore:rules

Deploy manual do site (opcional)

O deploy é automático pelo GitHub Actions. Para deploy manual:

cd web && npm run build
cp -r dist ../firebase/dist
cd ../firebase && firebase deploy --only hosting

⚙️ 5. GitHub Actions (CI/CD)

Para que o deploy automático funcione, configure os Secrets no GitHub:

Acesse: github.com/victoralbertoms-wq/rede-servicos-comunitarios → Settings → Secrets → Actions

FIREBASE_TOKENgerado com firebase login:ci

Gerar FIREBASE_TOKEN

firebase login:ci

Copie o token exibido e adicione como secret FIREBASE_TOKEN.

Gerar FIREBASE_SERVICE_ACCOUNT

  1. Acesse Firebase Console → Configurações do projeto → Contas de serviço
  2. Clique em "Gerar nova chave privada"
  3. Copie o conteúdo do JSON e cole no secret

🛡️ 6. Tornar Usuário Admin

Após criar a primeira conta no sistema:

1

Acesse o Firebase Console → Firestore Database → coleção users

2

Abra o documento do seu usuário

3

Edite o campo role para "admin" e salve

4
✅ Agora você terá acesso ao Painel Admin em /admin

⚡ Resumo de Comandos

Todos os comandos em sequência

# Clonar
git clone https://github.com/victoralbertoms-wq/rede-servicos-comunitarios.git
cd rede-servicos-comunitarios

# Web — criar web/.env com as credenciais, depois:
cd web && npm install && npm run dev

# Mobile — criar mobile/.env com as credenciais, depois:
cd mobile && npm install && npx expo start

# Regras Firestore
cd firebase && firebase deploy --only firestore:rules

# Publicar mudanças (deploy automático)
git add . && git commit -m "mensagem" && git push
