⚡ Comandos Rápidos

Rodar web local
cd web && npm run dev
Rodar mobile
cd mobile && npx expo start
Deploy (automático)
git add . && git commit -m "msg" && git push
Regras Firestore
cd firebase && firebase deploy --only firestore:rules
Login Firebase CLI
firebase login
Build produção
cd web && npm run build
