💡 O que é o sistema

A Rede de Serviços Comunitários é uma plataforma digital que conecta membros de comunidades a profissionais e empresas locais. Funciona como um diretório comunitário onde qualquer pessoa pode cadastrar seus serviços ou empresa, participar de comunidades e encontrar o que precisa.

🏘️ Comunidades

  • ✅ Listar todas as comunidades ativas com busca
  • ✅ Criar nova comunidade (apenas admin global)
  • ✅ Ver detalhe com seus serviços e empresas separados por aba
  • ✅ Editar nome, descrição, senha, logo e foto de capa (criador ou admin)
  • ✅ Remover logo ou foto de capa individualmente
  • ✅ Excluir comunidade (criador ou admin)
  • ✅ Entrar em comunidade com ou sem senha
  • ✅ Cadastrar serviço/empresa diretamente vinculado à comunidade

🔧 Serviços

  • ✅ Listar serviços aprovados com busca por nome, categoria, cidade
  • ✅ Filtro por categoria
  • ✅ Cadastrar novo serviço (qualquer usuário autenticado)
  • ✅ Ver detalhe completo: foto, contatos, avaliações, localização
  • ✅ Editar serviço — todos os campos + foto (dono ou admin)
  • ✅ Remover foto sem precisar enviar nova
  • ✅ Excluir serviço (dono ou admin)
  • ✅ Favoritar serviço
  • ✅ Entrar em contato por telefone, WhatsApp, e-mail, site

🏢 Empresas

  • ✅ Listar empresas aprovadas com busca e filtro por categoria

  • ✅ Ver detalhe completo: logo, foto de capa, contatos, avaliações
  • ✅ Editar empresa — todos os campos (dono ou admin)
  • ✅ Remover logo ou foto de capa individualmente
  • ✅ Excluir empresa (dono ou admin)

⭐ Avaliações

Qualquer usuário autenticado pode avaliar serviços e empresas com nota de 1 a 5 estrelas e comentário. A média é calculada automaticamente e exibida no card e na página de detalhe.

💬 Mensagens

Chat direto entre usuários, acessível pelo botão "Mensagem" na página de qualquer serviço ou empresa. Conversas ficam salvas no Firestore.

🔄 Fluxo de Aprovação

Serviços e empresas passam por aprovação antes de aparecer publicamente:

Usuário cadastra serviço/empresa
↓
pending   Aguardando aprovação
↓
Admin acessa /admin e aprova
↓
approved   Aparece na listagem pública

Nota: O próprio dono do cadastro sempre vê seu item, mesmo enquanto pendente.

🗄️ Estrutura de Dados (Firestore)

Coleção: users

{
  displayName: string,
  email: string,
  photoURL: string,
  role: 'user' | 'admin',
  communityId: string,
  favorites: { services: [], companies: [] },
  createdAt: timestamp
}

Coleção: communities

{
  name: string,
  description: string,
  password: string,       // vazio = comunidade aberta
  logoURL: string,
  photoURL: string,
  adminIds: [uid],        // quem pode editar/excluir
  memberCount: number,
  isActive: boolean,
  createdAt: timestamp
}

Coleção: services

{
  name, category, specialty, description,
  phone, whatsapp, email, website, instagram, facebook,
  city, state, address, workingHours,
  photoURL: string,
  communityId: string,
  userId: string,         // dono do cadastro
  status: 'pending' | 'approved' | 'rejected',
  rating: number,
  reviewCount: number,
  createdAt: timestamp
}

Coleção: reviews

{
  targetId: string,       // id do serviço ou empresa
  targetType: 'service' | 'company',
  userId, userName,
  rating: number,         // 1 a 5
  comment: string,
  createdAt: timestamp
}
